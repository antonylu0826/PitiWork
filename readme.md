# PotiWork
## PotiWork 是一個基於 React 和 TypeScript 的開發環境，並以 Keycloak 做為身份驗證機制，以 DevExpress XAF Web API Service 為後端，旨在提供一個快速、現代化的前端開發體驗。

- 建立 Devias Kit 範本
    
    ```bash
    git clone https://github.com/devias-io/material-kit-react.git
    cd material-kit-react
    npm install
    npm run dev
    ```
    

- 安裝全域環境的 Gemini CLI
    
    ```bash
    npm install -g @google/gemini-cli
    ```
    

- 在 react 目錄下啟動 Gemini CLI
    
    ```bash
    gemini
    ```
    

- gemini 初始提示詞
    
    ```bash
    這個目錄是一個 React+Typescript, 使用Devias Kit範本
    ```
